Registering a Domain

This section of the website is dedicated to exploring the steps necessary to register and set up a domain name for the website.



1. Purchase domain name

Domain names are overseen by organizations such as ICANN(Internet Corporation for Assigned Names and Numbers) and nation-specific bodies. Entities known as domain registrars (eg. GoDaddy and namecheap) are given accreditation by these bodies, authorizing them to register domain names on behalf of individuals and companies. To purchase a domain name, an entity needs to visit a website belonging to one such registrar and type in the desired domain name. The registrar then checks for the availability and validity of the domain name. If valid, then the entity may lease the domain name for a period of time. Domain names must be renewed regularly, depending on the purchase option.

2. Register for a web hosting service

When initially purchased, the domain will be empty. The domain owner must then register for a web hosting service, such as host gator. The domain owner must inform the hosting service of the domain name. Once entered, the hosting service will allow the domain owner a portion of their server space and the domain owner is given a namespace associated with their domain.

Many web hosting services are also accredited domain registrars and vice versa. It is thus possible to host a website with the same entity that registered a domain.

3. Connect the web host to the web domain

The domain owner must then return to the domain registrar website. They must associate their domain name with the associated namespace. This is not an instantaneous process, and the domain owner must wait for a few hours to a few days before local domain name servers register the domain and associate the domains with their associated namespaces.

4. Upload files to the web hosting service

The web hosting service will have the option to upload files. The necessary html and CSS files may be uploaded to the service, and the website will show in the required domain.
